## Método 1: Abrir Directamente en el Navegador

### Opción A: Doble clic
1. Ve a la carpeta `d:/PRUEBA2`
2. Busca el archivo `index.html`
3. Haz **doble clic** en `index.html`
4. Se abrirá automáticamente en tu navegador predeterminado

### Opción B: Clic derecho
1. Ve a la carpeta `d:/PRUEBA2`
2. Haz **clic derecho** en `index.html`
3. Selecciona **"Abrir con"** → **"Navegador web"** (Chrome, Firefox, Edge, etc.)

### Opción C: Arrastrar y soltar
1. Abre tu navegador web (Chrome, Firefox, Edge)
2. Arrastra el archivo `index.html` desde la carpeta hacia la ventana del navegador
3. Suelta el archivo y se abrirá automáticamente

## Método 2: Desde el Navegador

1. Abre tu navegador web
2. En la barra de direcciones, escribe:
   ```
   file:///d:/PRUEBA2/index.html
   ```
3. Presiona Enter

## Método 3: Usando Live Server (Recomendado para desarrollo)

Si tienes **Visual Studio Code**:

1. Instala la extensión **"Live Server"**
2. Abre la carpeta `d:/PRUEBA2` en VS Code
3. Haz clic derecho en `index.html`
4. Selecciona **"Open with Live Server"**
5. Se abrirá en `http://localhost:5500`

## 🔧 Solución de Problemas

### Si no se abre el archivo:
1. Verifica que tengas un navegador web instalado
2. Intenta con otro navegador
3. Verifica que el archivo `index.html` esté en la carpeta correcta

### Si aparecen errores de JavaScript:
1. Asegúrate de tener conexión a internet (para Firebase)
2. Verifica que todos los archivos estén en la misma carpeta
3. Intenta refrescar la página (F5)

### Si no funciona Firebase:
1. Verifica tu conexión a internet
2. Los datos se guardan localmente si no hay conexión
3. Al reconectarse, los datos se sincronizarán automáticamente
